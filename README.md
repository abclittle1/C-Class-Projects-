This assignment focused on fundamental C++ programming concepts including 
user input, mathematical functions, and control flow.

**Tools:** Visual Studio

**Projects:**
- Triangle classifier that identifies triangle type and calculates area using Heron's Formula
- Cylinder calculator that computes surface area and volume from user inputs
- Wind chill calculator that outputs a temperature/wind speed table using the NWS Wind Chill Index formula
